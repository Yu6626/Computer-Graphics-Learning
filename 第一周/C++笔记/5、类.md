


# 1. 类的基本结构

```cpp

class ClassName {
public:
    // 成员函数（方法）
    void setValue(int v) { value = v; }
    int getValue() const { return value; }
private:
    // 成员变量（数据）
    int value;
};
```


- **成员变量**：类的数据，通常放在 `private` 区域，对外隐藏。
    
- **成员函数**：操作数据的函数，可放在 `public` 区域供外部调用。
    
- **访问控制**：`public`（任何地方都可访问）、`private`（仅类内部或友元可访问）、`protected`（类内部及派生类可访问）。

# 2. 常用关键字

## 2.1 访问控制

- **`public`**：公开访问，类外部可直接使用。
    
- **`private`**：私有访问，仅类内部可访问（默认访问级别）。
    
- **`protected`**：受保护访问，类内部及派生类可访问。
    

## 2.2 特殊成员函数

- **`class` / `struct`**：`struct` 的成员默认是 `public`，`class` 默认是 `private`。功能上基本等价。
    
- **构造函数**：与类同名，无返回值。可重载，用于初始化对象。
    
    - `ClassName();` // 默认构造函数
        
    - `ClassName(int x);` // 带参构造函数
        
    - `ClassName(const ClassName& other);` // 拷贝构造函数
        
    - `ClassName(ClassName&& other);` // 移动构造函数（C++11）
        
- **析构函数**：`~ClassName()`，无参无返回值，对象销毁时自动调用。
    
- **赋值运算符**：`operator=`，用于对象赋值。
    
    - `ClassName& operator=(const ClassName& other);` // 拷贝赋值
        
    - `ClassName& operator=(ClassName&& other);` // 移动赋值（C++11）
        

## 2.3 对象与指针

- **`this`**：指向当前对象的指针，可在成员函数内使用。
    
- **`const`**：修饰成员函数时，表示该函数不会修改对象的数据成员（如 `int getValue() const;`）。
    

## 2.4 静态成员

- **`static`**：
    
    - 静态成员变量：属于类本身，所有对象共享一份，需在类外单独定义。
        
    - 静态成员函数：只能访问静态成员，通过类名直接调用。
        

## 2.5 继承与多态

- **继承**：`class Derived : public Base`（基类访问方式可为 `public`、`protected`、`private`）。
    
- **`virtual`**：
    
    - 虚函数：允许派生类重写，实现多态。
        
    - 纯虚函数：`virtual void func() = 0;`，使类成为抽象类，不能实例化。
        
- **`override`**（C++11）：显式声明重写基类虚函数，编译器会检查。
    
- **`final`**（C++11）：阻止类被继承，或阻止虚函数被重写。
    

## 2.6 友元

- **`friend`**：允许非成员函数或其他类访问本类的私有成员。
    

## 2.7 其他关键字

- **`explicit`**：阻止构造函数或转换运算符的隐式调用，通常用于单参数构造函数。
    
- **`mutable`**：允许在 `const` 成员函数中修改被 `mutable` 修饰的成员变量（通常用于缓存、互斥锁等）。
    
- **`constexpr`**（C++11）：使对象或函数能在编译期求值，可用于定义编译期常量类。
    
- **`noexcept`**（C++11）：声明函数不会抛出异常。

# 3. 常见结构与模式

## 3.1 类的定义与分离编译

通常将类声明放在头文件（`.h`），成员函数定义放在源文件（`.cpp`）中。


```cpp

// MyClass.h
class MyClass {
public:
    void doSomething();
private:
    int data;
};
// MyClass.cpp
#include "MyClass.h"
void MyClass::doSomething() {
    // 实现
}
```

## 3.2 初始化列表

构造函数使用初始化列表直接初始化成员变量，效率更高：

```cpp

MyClass(int x, double y) : data(x), value(y) {}
```

### 3.3 继承与虚函数表

派生类通过 `virtual` 函数实现多态，运行时根据实际对象类型调用对应函数。

```cpp

class Shape {
public:
    virtual void draw() const = 0;  // 纯虚函数
};
class Circle : public Shape {
public:
    void draw() const override { /* ... */ }
};
```

### 3.4 模板类

类也可以是模板，实现泛型编程。

```cpp

template<typename T>
class Box {
    T content;
public:
    void set(const T& val) { content = val; }
};
```
