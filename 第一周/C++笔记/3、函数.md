# 函数重载

- 多个函数同名，但是每个函数的形参类型和个数有所差异
- 在使用函数重载时，需要函数与函数之间可以被区分

# 函数模板

- 通常在函数定义前面加上`template<typename T>`或是`template<int N>`，两者可以组合，之间用逗号隔开，前者为类型参数，后者为非类型参数（如整数、枚举、指针和引用等）
- 在使用过程中，函数模板会根据输入值进行实例化赋予类型参数和或非类型参数实际值
- 除去直接传值的方法，还存在一个显式调用，即在调用时，在函数名后括号前添加一个`<（类型或者参数）>`可以把类型强制定义并将值传递

# 函数指针

- 定义的模板为`返回值类型 (*指针名)(形参类型表)`
- 在传参时，只需要写函数就可以了，即取地址符号可省
- 可以使得函数可以被模式化地调用，函数地址作为函数的参数被传入，一个函数可以视情况调用不同函数
